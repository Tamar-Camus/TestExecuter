In this section you can set or override your properties.
If you want to set/override a property you must check the checkbox next to it.
If you click on the "Fetch Properties From File" button it will fill the properties bellow from the properties file.
If you click on the "Show/Hide Available Fields" button it will show/hide all the fields used in the tests property you defined in the properties file.
Next to each available field, for "Show Fields", appears (howManyTestsHasThisField/amountOfTests).
